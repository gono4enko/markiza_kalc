# Журнал проекта: калькулятор маркиз

Источник анализа: `calculator_awning_v2_16.html` (одностраничное приложение: HTML + встроенные CSS и JavaScript).

**Как вести журнал:** при каждом значимом изменении добавляйте запись в раздел «Хронология» и обновляйте статусы задач в «Бэклоге».

---

## 1. Назначение продукта

Веб-калькулятор для расчёта стоимости маркиз (локтевая, витринная, вертикальная ZIP) с детализацией позиций, блоком заявки (мессенджеры, обратный звонок), отправкой лида в Telegram и на email через FormSubmit, целью Яндекс.Метрики.

---

## 2. Архитектура (зафиксированные решения)

| Решение | Обоснование |
|--------|-------------|
| Один файл `.html` | Простое встраивание на Tilda/статический хостинг без сборки. |
| Глобальные функции в `window` | Вызовы из атрибутов HTML (`onclick`, `onchange`): `updateConfigPreview`, `updateFabricPreview`, `tcScroll`, `submitLead`. |
| Цены в объектах-матрицах | `PRICES_OPEN`, `PRICES_SEMI`, `PRICES_CASSETTE` — ширина × вылет; `PRICES_G400` / `PRICES_G450` — витринные; `ZIP100` / `ZIP130` — ширина × высота. |
| Округление габаритов | `nextStd(v)`: шаг 0.5 м вверх для попадания в ячейку таблицы. |
| Курс / масштаб | `euroRate = 100`: базовые значения в матрицах умножаются на 100 для отображения в рублях (логика названа «евро», фактически — коэффициент к прайсу). |
| Надбавки от базы | Ткань и цвет каркаса считаются как доля от `base`; доставка/изготовление — 10% от суммы комплекта без монтажа; монтаж — отдельные формулы по типу. |
| Превью | Тип маркизы: карусель изображений в `AWNING_PREVIEWS`; конструкция: переключение `cfg-img` по `config` и `awning-type`; ткани: горизонтальный трек `FABRIC_DATA` (URL|||название). |
| Геолокация для текста заявки | `ipapi.co/json/` → `window._userCity` для WA/TG/Max. |

---

## 3. Входные параметры и ветвления UI

- **Тип маркизы:** `standard` | `storefront` | `zip` — переключает группы полей (вылет vs высота, ткань vs ткань ZIP, цвет каркаса, датчики/подсветка, опции `config`).
- **ZIP:** только электропривод; ручное отключено; `config` — ZIP 100 / ZIP 130.
- **Локтевая + кассетная:** мотор Decolife отключается в UI, принудительно сброс на Simu при выборе.
- **Витринная:** вылеты 0.8 / 1.0 / 1.4 м; нет опции «датчик ветровых колебаний» (только ветер+солнце или без).
